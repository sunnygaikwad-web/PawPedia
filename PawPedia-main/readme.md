# 🐾 PawPedia – Smart Pet Care & Animal Learning Platform

PawPedia is an interactive, modern web platform designed to help users **learn about animals, take better care of pets, and explore adoption opportunities**.

It combines features of:

* 📖 Encyclopedia (like Wikipedia)
* 🎓 Learning Hub (interactive modules + quizzes)
* 🐶 Pet Care Tracker (daily activities & health)
* 🤖 AI Chatbot (ask anything about animals)

---

## 🚀 Features

### 📚 Animal Encyclopedia

* Explore a wide range of animals dynamically
* Search and filter by categories (Mammals, Birds, Reptiles, etc.)
* Detailed pages with:

  * Images
  * Habitat, diet, lifespan
  * Fun facts

---

### 🎓 Learning Hub

* Multiple learning topics:

  * Mammals, Birds, Marine Life, Ecosystems, etc.
* Interactive learning pages with:

  * Real images
  * Key concepts & explanations
  * Fun facts

#### 🧠 Dynamic Quiz System

* Generate **unique MCQs using APIs**
* No repeated questions
* Instant feedback + score tracking

---

### 🐾 Pet Care System

* Care guides for different pets
* Daily activity tracker:

  * Feeding
  * Grooming
  * Walking
* Health & vaccination reminders
* Simple smart suggestions

---

### 🐶 Adoption Section

* Browse pets available for adoption
* Filter by type and preferences
* View pet details and adoption options

---

### 🤖 AI Chatbot

* Ask anything about animals or pet care
* Get detailed and creative responses
* Example:

  * “Tell me about lions”
  * “How to take care of a dog?”

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* APIs Used:

  * Animal Data APIs (Wikipedia / Zoo APIs)
  * Image APIs (Unsplash / Pexels)
  * AI / Quiz Generation APIs
